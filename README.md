# SFULL - Proyecto Full Stack

## Estructura del proyecto

- `client/` - Frontend (React + TypeScript + Vite)
- `server/` - Backend (Node.js + Express)

## Instalación

### Cliente

```bash
cd client
npm install
npm run dev
```

### Servidor

```bash
cd server
npm install
npm run dev
```

## Variables de entorno

Configura los archivos `.env` en cada carpeta según `.env.example`
